# revops-automation-toolkit
Automations for common operational headaches: board governance publishing, email blast campaigns, group assignment tracking. Built with Google Apps Script.

Problem: Charter schools waste hours on manual board reports, email campaigns, roster management

Solution: Automated tools using Google Apps Script that integrate with Google Workspace
This google app script allows for any Wordpress site to be automated to upload any docx and pdf to a governance or compliance page and be listed in a searchable and clean format.

Features: Board doc auto-publisher, email blast system, group assignment tracker
Impact: Saved 20+ hours/month, enabled data-driven decision making

Tech Stack: Google Apps Script, Google Sheets API, Gmail API
